# ca trust binder

## Prerequisites
- `cert-manager`
  - v1.12.2
  - https://charts.jetstack.io/charts/cert-manager-v1.12.2.tgz
  - install with CRDs
- `trust-manager`
  - v0.5.0
  - https://charts.jetstack.io/charts/trust-manager-v0.5.0.tgz
  - install with CRDs
- `gatekeeper`
  - 3.12.0
  - https://open-policy-agent.github.io/gatekeeper/charts/gatekeeper-3.12.0.tgz
